【问题 ID】: stress_13
【测试问题】: 在什么极端情况下，非农业建设占用耕地可以免除实施耕作层剥离的义务？
【预期关键词】: ['严重破坏', '严重污染', '无法剥离再利用', '县级人民政府同意']
【失败原因】: 召回为空或未命中核心词
【大模型生成的 Cypher】:
MATCH (node:Article)
WHERE node.full_text CONTAINS '耕地' AND node.full_text CONTAINS '耕作层剥离' AND (node.full_text CONTAINS '免除' OR node.full_text CONTAINS '不宜' OR node.full_text CONTAINS '例外')
RETURN node.number AS article_number, node.full_text AS content
UNION
MATCH (node:Obligation)
WHERE node.description CONTAINS '耕地' AND node.description CONTAINS '耕作层剥离' AND (node.description CONTAINS '免除' OR node.description CONTAINS '不宜')
RETURN node.source_article AS article_number, node.description AS content
